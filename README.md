# Code Alpha Tasks 

Welcome to the **Code Alpha Tasks** repository! This collection showcases a series of data science and machine learning projects completed as part of the Code Alpha internship program. Each folder contains a standalone project with its own dataset, code, and analysis.

##  Project Overviews

### 1.  Iris Flower Classification
- **Goal:** Classify iris flowers into species using supervised machine learning.
- **Techniques:** Logistic Regression, Decision Trees, KNN
- **Tools:** Python, scikit-learn, Matplotlib, Seaborn
- **Highlights:** Exploratory Data Analysis (EDA), model evaluation, and accuracy comparison.

### 2.  Sales Price Prediction
- **Goal:** Predict housing prices based on various features.
- **Techniques:** Linear Regression, Random Forest, Gradient Boosting
- **Tools:** Python, Pandas, scikit-learn, XGBoost
- **Highlights:** Feature engineering, model tuning, and performance metrics.

### 3.  Unemployment Analysis in India
- **Goal:** Analyze unemployment trends across Indian states.
- **Techniques:** Data visualization, time-series analysis
- **Tools:** Python, Pandas, Matplotlib, Plotly
- **Highlights:** Regional comparisons, interactive plots, and policy insights.
